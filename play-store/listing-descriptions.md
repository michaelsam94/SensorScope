## Full description (4000 chars max)
SensorScope is a focused hardware diagnostics toolkit for Android devices. It helps you inspect sensor availability, monitor live readings, review battery telemetry, run controlled thermal checks, and export sensor snapshots for deeper analysis.

Use SensorScope to:

- View a real-time sensor health dashboard grouped by motion, environment, position, and unknown hardware categories.
- Inspect individual sensor metadata including vendor, version, range, resolution, power draw, and minimum delay.
- Plot live sensor streams with calibrated X, Y, and Z readouts.
- Capture and clear calibration baselines for supported sensors.
- Monitor battery level, voltage, current flow, temperature, charging state, technology, and anomaly warnings.
- Run a controlled thermal stress sandbox with safety limits, stop control, result summaries, and historical logs.
- Export recent sensor readings as CSV snapshots for sharing or analysis.
- Preview diagnostic reports from current sensor, battery, and thermal data.

SensorScope is built for technicians, developers, QA teams, and power users who want a clear view into the hardware signals their Android device exposes.

Privacy note: SensorScope reads local device hardware telemetry and stores diagnostic baselines/logs on the device. Exported CSV snapshots are only shared when you choose to share them.
